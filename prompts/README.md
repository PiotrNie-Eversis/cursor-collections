# Prompt library (canonical location)

The **Cursor** prompt catalog (`eversis-*.md`) lives under **`website/docs/prompts/public/`** and **`website/docs/prompts/internal/`** — same files power the docs site and Agent attachments.

Attach in Cursor with `@website/docs/prompts/public/eversis-implement.md` (and similar paths). See [documentation/cursor-collection.md](../documentation/cursor-collection.md) and [website/docs/prompts/overview.md](../website/docs/prompts/overview.md).

This `prompts/` folder is intentionally **empty** of Markdown bodies in this repository to avoid duplicating the catalog.
